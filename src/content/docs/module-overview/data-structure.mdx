The **Data Structure Operations** module in NumLex provides classes and functions for creating and manipulating basic data structures like stacks and queues.

## Classes and Functions

### Simple Stack (`SimpleStack`)

This class allows for basic stack operations like adding, removing, updating, and searching for items.

- **`add(item)`**: Adds an item to the stack if there is space.

  <Code code={`stack = simp_stk(5)\nstack.add(10)`} lang="python"></Code>

- **`delete()`**: Removes the top item from the stack.

  <Code code={`stack.delete()`} lang="python"></Code>

- **`update(old_item, new_item)`**: Updates an existing item in the stack with a new value.

  <Code code={`stack.update(10, 20)`} lang="python"></Code>

- **`display()`**: Returns the current items in the stack.

  <Code code={`items = stack.display()`} lang="python"></Code>

- **`count()`**: Returns the number of items currently in the stack.

  <Code code={`count = stack.count()`} lang="python"></Code>

- **`search(item)`**: Searches for an item in the stack and returns `True` if found, otherwise `False`.

  <Code code={`found = stack.search(20)`} lang="python"></Code>

- **`sort()`**: Sorts the items in the stack.

  <Code code={`stack.sort()`} lang="python"></Code>

- **`isEmpty()`**: Checks if the stack is empty.

  <Code code={`empty = stack.isEmpty()`} lang="python"></Code>

- **`isFull()`**: Checks if the stack is full.

  <Code code={`full = stack.isFull()`} lang="python"></Code>
