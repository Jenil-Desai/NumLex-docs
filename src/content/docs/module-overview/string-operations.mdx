The **String Operations** module in NumLex provides various utilities for working with strings, including character checks, palindrome detection, and character counting.

## Functions

- **`is_vowel(char)`**: Checks if a character is a vowel.

  <Code code={`check = is_vowel('a')`} lang="python"></Code>

- **`is_consonant(char)`**: Checks if a character is a consonant.

  <Code code={`check = is_consonant('b')`} lang="python"></Code>

- **`find_char(string, char)`**: Searches for a character in a string.

  <Code code={`found = find_char('hello', 'e')`} lang="python"></Code>

- **`is_palindrome(string)`**: Checks if a string is a palindrome.

  <Code code={`palin = is_palindrome('radar')`} lang="python"></Code>

- **`cnt_char(string)`**: Counts the occurrences of each character in a string.

  <Code code={`count = cnt_char('hello')`} lang="python"></Code>
