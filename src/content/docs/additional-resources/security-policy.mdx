## Reporting a Vulnerability

We take security seriously and appreciate your efforts to responsibly disclose vulnerabilities.

### How to Report

If you find a security vulnerability, please report it to our security team via email at `jenildev91@gmail.com`. Provide a detailed description of the issue, including:

- Affected versions
- Steps to reproduce the vulnerability
- Potential impact
- Any possible fixes

### Response Time

You can expect an initial response within 72 hours of your report. We will work with you to understand the issue and provide a timely fix.

### After Reporting

Once the vulnerability is confirmed, we will take the following steps:

1. **Issue a Fix**: A patch will be developed to address the vulnerability.
2. **Notify the Community**: We will notify users through a security advisory and update the affected versions.
3. **Public Disclosure**: After the patch is released, we will disclose the vulnerability details in the changelog.

Your security report is greatly appreciated, and we thank you for helping to keep NumLex safe!
