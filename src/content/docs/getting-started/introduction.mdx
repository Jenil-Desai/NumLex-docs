# Welcome to NumLex

NumLex is a powerful and versatile Python package designed to simplify your mathematical and string manipulation tasks. Whether you're working on geometry, arithmetic, probability, statistics, or even string and number operations, NumLex has got you covered.

## Get Started

To start using NumLex, follow the steps outlined in the **Installation** section.

For detailed information on each module and the functions available, check out the **Modules** section.

Need help or want to contribute? Visit the **Contributing** and **FAQ** sections.
